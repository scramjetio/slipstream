# Security Policy

## Supported Versions

Currently, only the `main` branch and the latest published release are supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please DO NOT open a public issue. Instead, send an email to security@scramjet.io. 

We will respond within 48 hours to acknowledge the report and outline the next steps for mitigation. We ask that you maintain confidentiality until we have had a reasonable timeframe to publish a fix.
