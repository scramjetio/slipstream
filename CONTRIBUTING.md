# Contributing to Scramjet Open Source

First off, thank you for considering contributing to our open-source projects! It's people like you that make Scramjet a great ecosystem.

## How Can I Contribute?

### Reporting Bugs
Bugs are tracked as GitHub issues. When you create an issue, please use the provided template and include as many details as possible to help us reproduce the issue.

### Suggesting Enhancements
Enhancement suggestions are also tracked as GitHub issues. Please use the Feature Request template and provide a clear description of the new feature or improvement.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes (`npm run check` or `npm test`).
4. Make sure your code lints.
5. Issue that pull request!

## Code of Conduct
This project and everyone participating in it is governed by the [Scramjet Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
